# Claude Code Skills

A collection of custom [Claude Code](https://docs.anthropic.com/en/docs/claude-code) skills for n8n workflow development and project planning.

## Skills

| Skill | Description |
|-------|-------------|
| **n8n-code-javascript** | Write JavaScript code in n8n Code nodes |
| **n8n-code-python** | Write Python code in n8n Code nodes |
| **n8n-expression-syntax** | Validate n8n expression syntax and fix common errors |
| **n8n-mcp-tools-expert** | Expert guide for using n8n-mcp MCP tools effectively |
| **n8n-node-configuration** | Operation-aware node configuration guidance |
| **n8n-validation-expert** | Interpret validation errors and guide fixing them |
| **n8n-workflow-patterns** | Proven workflow architectural patterns from real n8n workflows |
| **project-planner** | Break down complex projects into actionable tasks |

## Installation

Clone this repo into your Claude Code skills directory:

```bash
git clone https://github.com/noamblatt1/claude-code-skills.git
cp -r claude-code-skills/* ~/.claude/skills/
```
